# Cursor Hackathon Sudbury 2026

**Build the North**

Welcome to the official participant resource page for Cursor Hackathon Sudbury 2026.

This page contains the main information participants need before and during the hackathon, including account setup, tool credits, API access, Devpost submission, and useful links.

## Event

**Date:** Saturday  
**Registration:** https://luma.com/cursor-sudbury-canada

## Start Here

Before the event, please complete these steps:

1. Install Cursor: https://cursor.com/downloads
2. Create a Cursor account: https://cursor.com
3. Create a Lovable account: https://lovable.dev
4. Create an Apify account: https://apify.com
5. Create a Valsea / Valencia API account: https://valsea.ai/dashboard/billing
6. Create or sign in to Devpost for project submission: https://devpost.com

## What to Bring

Please bring:

- Laptop
- Charger
- Cursor installed
- Devpost account or ability to create one
- Any project idea, dataset, or problem you want to explore

## Main Tools

During the hackathon, participants may use:

- Cursor for AI-assisted coding
- Devpost for final project submission
- Lovable for quickly building web apps and prototypes
- Apify for web automation, scraping, and data workflows
- Valsea / Valencia API for voice, audio, or related AI features
