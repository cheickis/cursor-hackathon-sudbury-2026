# Credits and API Access

This page explains how to activate available credits and API access for the hackathon.

---

## Valsea / Valencia API

Follow these steps:

1. Go to [Valsea Billing](https://valsea.ai/dashboard/billing).
2. Sign in or create an account.
3. On the Billing page, choose the **Free** plan.
4. The Free plan includes **55 minutes one-time**.
5. Go to **API Keys** in the sidebar.
6. Create or copy your API key.
7. Use that key in your hackathon project with the Valencia API.
8. Track usage under **Billing** or **Analytics**.

Note: the Free plan has a hard cap, so requests stop once credits are exhausted.

## Lovable Credits

Lovable gives participants access to **Pro Plan 1 with 100 credits** at no cost.

### How it Works

- The access gives participants **Pro Plan 1** with **100 credits**.
- It unlocks all Pro Plan 1 features for the duration of the plan.
- The access must be redeemed by the end of the event.
- The event access should not be shared with anyone outside the event.

### How to Apply the Event Access

1. Go to [Lovable](https://lovable.dev).
2. If you do not have an account, click **Get started** and create an account.
3. If you already have an account, go to **Settings → Plans & Credits**.
4. Select **Pro Plan 1** with **100 credits**.
5. Make sure to choose the **monthly** plan.
6. At checkout, enter the event discount code.
7. Complete checkout.

If you already have a paid Lovable account, create a new workspace to redeem the event access.

Please note: you may be asked to enter payment details, but you will have one month to cancel. You can cancel the account as soon as it is activated.

### Event Discount Code

The Lovable event discount code will be shared with registered participants during the event.

Please do not share it outside the event.

## Apify Credits

Use Apify for web scraping, automation, and data workflows.

### Create an Apify Account

1. Go to [Apify](https://apify.com).
2. Create an account or sign in.
3. Open the Apify Console.

### Apply an Apify Promo Code

If the organizers provide an Apify promo code:

1. Go to **Billing → Subscription**.
2. Scroll to **Special offers**.
3. Click **Add promo code**.
4. Enter the event promo code.
5. Save and confirm the code was applied.

### Apify Event Code

The Apify event code will be shared with registered participants if available.

Please do not share it outside the event.

### Create an Apify API Token

1. In Apify Console, go to **Settings**.
2. Open **API & Integrations** or **Integrations**.
3. Copy your API token.
4. Store the token safely in your project.

Do not share your token publicly.
